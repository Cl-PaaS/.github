## Git Convention


Hotfix

- 치명적인 버그 긴급 수정



Chore

- 빌드 업무, 패키지 매니저, 패키지 관리자 구성 등 업데이트 / 프로덕션 코드 변경 없음



Comment

- 주석 추가 및 변경



Docs

- 문서 수정



Feat

- 신규 기능 추가



Fix

- 버그 수정



Refactor

- 프로덕션 코드 리팩토링



Style

- 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우



Test

- 테스트 코드 추가 / 프로덕션 코드 변경 없음
